2. Installing Karrigell

First you need to install Python 2.3 or above : go to the Python language website and follow the installation instructions for your operating system

Decompress the archive you have downloaded (.zip, .tgz or .tar.bz2) in a directory such as ; then go to the directory called Karrigell-2.4.0, we'll call it the Server Directory

To launch the program, open a console window, go to the Server Directory and run

(server dir)>Karrigell-2.4.0>python Karrigell.py
You should read the message :

Karrigell 2.4.0 running on port 80
Press Ctrl+C to stop

Type in http://localhost in the address field of your browser

If this is your first connection, your will be prompted to enter a login and a password for the site administrator :

Create a login/password for administrator

Login
Password
Confirm password

Once they are set, you can see the home page

  1  """Classes to generate HTML in Python 
  2   
  3  The HTMLTags module defines a class for all the valid HTML tags, written in 
  4   uppercase letters. To create a piece of HTML, the general syntax is : 
  5   
  6      t = TAG(innerHTML, key1=val1,key2=val2,...) 
  7   
  8  so that "print t" results in : 
  9   
 10      <TAG key1="val1" key2="val2" ...>innerHTML</TAG> 
 11   
 12  For instance : 
 13   
 14      print A('bar', href="foo") ==> <A href="foo">bar</A> 
 15   
 16  To generate HTML attributes without value, give them the value True : 
 17   
 18      print OPTION('foo',SELECTED=True,value=5) ==>  
 19              <OPTION value="5" SELECTED> 
 20   
 21  For non-closing tags such as <IMG> or <BR>, the print statement does not  
 22  generate the closing tag 
 23   
 24  The innerHTML argument can be an instance of an HTML class, so that you can nest  
 25  tags, like this : 
 26   
 27      print B(I('foo')) ==> <B><I>foo</I></B> 
 28   
 29  Instances of the HTML classes support the addition : 
 30   
 31      print B('bar')+INPUT(name="bar") ==> <B>bar</B><INPUT name="bar"> 
 32   
 33  and also repetition : 
 34   
 35      print TH('&nbsp')*3 ==> <TD>&nbsp;</TD><TD>&nbsp;</TD><TD>&nbsp;</TD> 
 36   
 37  If you have a list (or any iterable) of instances, you can't concatenate the items with  
 38  sum(instanceList) because sum takes only numbers as arguments. So there is a  
 39  function called Sum which will do the job : 
 40   
 41      Sum( TR(TD(i)+TD(i*i)) for i in range(100) ) 
 42   
 43  generates the rows of a table showing the squares of integers from 0 to 99 
 44   
 45  The innerHTML argument can be a string, but you can't concatenate a string and  
 46  an instance of an HTML class, like in : 
 47   
 48      H1('To be or ' + B('not to be')) 
 49   
 50  For this, use a class called TEXT, which will not generate any tag : 
 51   
 52      H1(TEXT('To be or ') + B('not to be')) 
 53   
 54   
 55  A simple document can be produced by : 
 56   
 57      print HTML( HEAD(TITLE('Test document')) + 
 58          BODY(H1('This is a test document')+ 
 59               TEXT('First line')+BR()+ 
 60               TEXT('Second line'))) 
 61   
 62  This will produce : 
 63   
 64      <HTML> 
 65      <HEAD> 
 66      <TITLE>Test document</TITLE> 
 67      </HEAD> 
 68      <BODY> 
 69      <H1>This is a test document</H1> 
 70      First line 
 71      <BR> 
 72      Second line 
 73      </BODY> 
 74      </HTML> 
 75   
 76  If the document is more complex it is more readable to create the elements  
 77  first, then to print the whole result in one instruction. For example : 
 78   
 79  stylesheet = LINK(rel="Stylesheet",href="doc.css") 
 80   
 81  head= HEAD(TITLE('Record collection')+stylesheet) 
 82  title = H1('My record collection') 
 83  rows = Sum ([TR(TD(rec.title,Class="title")+TD(rec.artist,Class="Artist"))  
 84      for rec in records]) 
 85  table = TABLE(TR(TH('Title')+TH('Artist')) + rows) 
 86   
 87  print HTML(head + BODY(title + table))

markLine(cdcfile, keyword)

source code 
搜索匹配函式:
Parameters:
  • cdcfile - *.cdc文件名，含路径
  • keyword - 搜索关键词，应该是utf-8 编码
Returns:
将所有匹配的行组织成 report 返回

Note: 粗略的在摘要文件行中匹配，并直接输出该行;为了E文的通用匹配，预先将关键词和行文字都变成大写的

markIni(cdcfile, keyword)

source code 
配置文件模式匹配函式:
Parameters:
  • cdcfile - *.cdc文件名，含路径
  • keyword - 搜索关键词，应该是utf-8 编码
Returns:
将所有匹配的行组织成 report 返回

Note: 将文件解析成配置对象树,然后进行匹配;为了E文的通用匹配，预先将关键词和行文字都变成大写的:

  • 如果在目录名上有匹配就记录目录,并停止挖掘
  • 如果在文件名上有匹配,则前缀目录名记录
